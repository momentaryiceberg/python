Basic Syntax:

  1. "import re" to import the regex module.
  2. "re.compile(pattern)" to compile a regex pattern.
  3. "re.match(pattern, string)" to match a pattern at the beginning of a string.
  4. "re.search(pattern, string)" to search for a pattern anywhere in a string.
  5. "re.findall(pattern, string)" to find all occurrences of a pattern in a string.
  6. "re.sub(pattern, replacement, string)" to substitute a pattern with a replacement in a string.

Special Characters:

  • .: Matches any single character except newline.
  • ^: Matches the start of a string.
  • $: Matches the end of a string.
  • *: Matches 0 or more occurrences of the preceding character.
  • +: Matches 1 or more occurrences of the preceding character.
  • ?: Matches 0 or 1 occurrence of the preceding character.
  • []: Matches any single character within the brackets.
  • |: Matches either the pattern before or after the pipe.
  • (): Groups patterns together.

Character Classes:

  • \d: Matches any digit (0-9).
  • \D: Matches any non-digit character.
  • \w: Matches any alphanumeric character (word character).
  • \W: Matches any non-alphanumeric character.
  • \s: Matches any whitespace character.
  • \S: Matches any non-whitespace character.

Quantifiers:

  • {n}: Matches exactly n occurrences of the preceding character.
  • {n,}: Matches n or more occurrences of the preceding character.
  • {n,m}: Matches between n and m occurrences of the preceding character.

Anchors:

  • \b: Matches a word boundary.
  • \B: Matches a non-word boundary.
  • ^: Matches the start of a string.
  • $: Matches the end of a string.

Flags:

  • re.IGNORECASE or re.I: Case-insensitive matching.
  • re.MULTILINE or re.M: Multiline matching.
  • re.DOTALL or re.S: Dot matches all, including newline.
  • re.VERBOSE or re.X: Verbose mode (allows comments and whitespace).

Examples:

Match a word:

pattern = r'\bword\b' re.findall(pattern, 'This is a word.')

Match an email:

pattern = r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b' re.findall(pattern, 'My email is example@example.com')

Replace a word:

pattern = r'\bword\b' replacement = 'new_word' re.sub(pattern, replacement, 'This is a word.')

Match a date:

pattern = r'\b\d{4}-\d{2}-\d{2}\b' re.findall(pattern, 'The date is 2022-01-01')

Match a phone number:

pattern = r'\b\d{3}-\d{3}-\d{4}\b' re.findall(pattern, 'My phone number is 123-456-7890') Remember to use raw strings (r'...') for regex patterns to avoid issues with backslashes. Also, practice using online regex testers like regex101.com to test your patterns.
